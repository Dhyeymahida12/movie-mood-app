# Movie Mood App 🎬

Movie Mood App is a modern Android application designed to recommend movies based on the user's current mood. Whether you're looking for a laugh, a thrill, or a cozy comfort film, this app curates the perfect cinematic match for your emotional state.

## 🚀 Features

* **Mood-Based Movie Recommendations:** Instantly find films tailored to match how you are feeling right now.
* **Database Integration:** Reliable data handling for a seamless browsing experience.
* **Admin Panel:** Built-in administrative tools to manage movie listings and application data.
* **Spoiler-Free Summaries:** Get the gist of a film without worrying about ruining the plot twists.
* **Movie Facts:** Discover interesting trivia and fun facts behind your favorite films.
* **Modern Cinematic UI:** A visually immersive interface designed for film enthusiasts.

## 🛠️ Technologies Used

* **Kotlin:** Main programming language for modern, concise, and safe Android development.
* **Android SDK:** Core framework utilized to build the mobile application architecture.
* **SQLite / Room Database:** Used for local data persistence, caching, and offline capability.

## 📦 Project Architecture

The app follows standard Android development guidelines, separating concerns for scalability and clean code management:
* **UI Layer:** Handles layouts, splash screens, and interactive user flows.
* **Data Layer (Room/SQLite):** Manages local tables, entity mappings, and database migrations.
